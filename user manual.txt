Electronics Tracker - User Manual

Welcome to the Electronics Tracker application! This manual will guide you through the basic setup and usage of the software.

1. Installation
   - Ensure Python 3.8 or higher is installed
   - Install required dependencies by running:
     pip install -r requirements.txt

2. Starting the Application
   - Run the main application file:
     python electronics_tracker.py

3. Basic Features
   - Add new electronics items to your inventory
   - Track item locations and status
   - View detailed information about each item
   - Generate inventory reports

4. Navigation
   - Use the menu system to access different features
   - Follow on-screen prompts for specific actions
